# SpringBoot-Scala g8 template

## Overall

- SpringBoot 3.0.4
  - PostgreSQL + Flyway
  - mustache
- Scala 2.13.10
  - scalatest
- Scala.js 1.13.0

```shell
sbt new kijuky/springboot-scala.g8
```

## Test

```shell
sbt g8Test
```

- not enabled hot reload
